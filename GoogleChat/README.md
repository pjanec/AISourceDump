# Extractor for google AI studion saved chat

 1. enable auto-save in  google ai studio chat so it gets saved to google disk
 2. in google disk, download the file as json
 3. use this utility to extract the conversion to a separate markdown file



# FAR integration

Add command to FAR user menu
 - Label:
 - Command: `python "C:\Utils\AICodeDump\extract_chat.py" "!.!" "!.!.md"`
